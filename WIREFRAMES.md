## Design Philosophy

### Banking-Grade Aesthetic
- **Clean & Minimal**: Stripe/Razorpay-inspired design
- **No Flashy UI**: Focus on functionality and clarity
- **Professional**: Suitable for CA firms and financial institutions
- **Accessible**: Clear typography, proper contrast
- **Responsive**: Works seamlessly on all screen sizes

### Color Palette
```
Primary:    #2563eb (Blue-600)
Success:    #16a34a (Green-600)
Warning:    #f59e0b (Amber-500)
Error:      #ef4444 (Red-500)
Background: #ffffff (White) / #f9fafb (Gray-50)
Text:       #111827 (Gray-900)
Border:     #e5e7eb (Gray-200)
```

### Typography
```
Font Family: Inter, system-ui, sans-serif
Headings:    600-700 weight
Body:        400-500 weight
Labels:      500 weight
```

## Responsive Design

### Desktop (> 1024px)
- Full sidebar visible
- Multi-column layouts
- Expanded charts and tables
- All features accessible

### Tablet (640px - 1024px)
- Collapsible sidebar
- 2-column layouts
- Responsive charts
- Touch-friendly controls

### Mobile (< 640px)
- Hidden sidebar (hamburger menu)
- Single-column layouts
- Stacked cards
- Mobile-optimized forms
- Bottom navigation bar

### Breakpoint Strategy

```css
/* Mobile First Approach */
.container {
  /* Mobile: < 640px */
  padding: 1rem;
}

@media (min-width: 640px) {
  /* Tablet */
  .container {
    padding: 1.5rem;
  }
}

@media (min-width: 1024px) {
  /* Desktop */
  .container {
    padding: 2rem;
  }
}
```
